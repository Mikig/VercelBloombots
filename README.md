FIELD WORK TIMER - Complete PWA Deployment Package
  
  To deploy on Vercel:
  1. Create a new folder on your computer called "field-work-timer"
  2. Create these 5 files inside that folder (copy each section below):
     - index.html
     - manifest.json
     - sw.js (service worker)
     - icon-192.png (create or use emoji)
     - icon-512.png (create or use emoji)
  3. Deploy to Vercel (drag the folder or connect via GitHub)
  
  File structure:
  field-work-timer/
  ├── index.html
  ├── manifest.json
  ├── sw.js
  ├── icon-192.png
  └── icon-512.png
